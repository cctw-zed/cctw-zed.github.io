Posted on 2019-12-30

hexo 搭建记录

记录是一件很重要的事情

安装依赖

  • Node.js（如果已经安装，确保版本不低于8.10。若进行下一步提示node错误建议卸载重装）
  • Git

安装hexo

使用命令行到命令行用户（或者所有命令前加上sudo）：

$ su root

安装命令：

$ npm install -g hexo-cli

但在国内使用npm下载速度奇慢，建议使用淘宝源cnpm：

$ npm install -g cnpm --registry=http://registry.npm.taobao.org

$ cnpm install -g hexo-cli

建站

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。(可能会很慢，要从GitHub下载文件。耐心等待。)

$ hexo init <folder>
$ cd <folder>

查看hexo版本

$ hexo -v

启动

$ hexo s

之后访问”localhost:4000”即可看到建成的博客。ctrl+v关闭hexo。

写博客

  • 生成文章,文章名为xxx.md，生成信息中有文章的地址。

$ hexo n 'xxx'

  • 使用编辑器编辑markdown文件

$ sudo vim xxx的地址

  • 上传文章

$ hexo clean
$ hexo s

部署到GitHub Page

  • 在自己的GitHub上新建一个仓库，名为：<github账户名>.github.io

  • 安装git部署插件

$ cnpm install --save hexo-deployer-git

  • 修改hexo配置文件

$ vim config.yml

部署

$ hexo d

  • 再访问<github账户名>.github.io即可访问到自己的博客。
